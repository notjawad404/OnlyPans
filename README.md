# MealCraft

A full-stack recipe sharing application built with React and Node.js that allows users to discover, create, and manage recipes through a clean and responsive interface. The application provides complete CRUD functionality, enabling users to browse recipes, view detailed instructions, and easily add, edit, or remove their own recipes. It uses a RESTful API with MongoDB for reliable data storage and retrieval.

## Tech Stack

**Frontend** — React, React Router, Tailwind CSS, Vite
**Backend** — Node.js, Express, MongoDB (Mongoose)

## Features

* Browse all recipes
* View a single recipe
* Add a new recipe
* Edit or delete existing recipes
